# Masseria Spina Wedding Venue Review

Interactive review page for [Masseria Spina Resort](https://www.masseriaspinaresort.com/) in Monopoli, Puglia, Italy.

## Live Site

**[View the interactive review page](https://brandonhawi.github.io/wedding-venue-review/)**

## Features

- Interactive D3.js flight path visualization (LAX → Rome → Bari)
- Budget calculator with guest count and menu tier options
- Photo gallery of venue spaces
- Google Maps driving directions from Bari Airport
- Responsive design
